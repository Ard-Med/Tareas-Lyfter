Cree un algoritmo que le pida 2 números al usuario, los guarde en dos variables distintas (primero y segundo) y los ordene de menor a mayor en dichas variables.
START
DEFINE primero = 0
DEFINE segundo = 0
DEFINE stored_value1 = 0
DEFINE stored_value2 = 0
PRINT "Ingresa dos numeros"
INPUT primero
INPUT segundo

IF (primero > segundo)
 stored_value1 = primero
 stored_value2 = segundo
 primero = stored_value2
 segundo = stored_value1

PRINT primero
PRINT segundo
END


Cree un algoritmo que le pida al usuario una velocidad en km/h y la convierta a m/s.
Recuerda que 1 km == 1000m y 1 hora == 60 minutos * 60 segundos.

START
DEFINE speed
DEFINE meter_seconds
PRINT "Enter a speed value"
INPUT speed

meter_seconds = speed * 1000 / 3600

PRINT "La velocidad en metros sobre segundo es de: "
PRINT meter_seconds
END
